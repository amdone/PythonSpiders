### 详细说明

#### acfun2.py

类似命令python3 acfun2.py <uid> ./ -g gd

uid就是up主的空间号，分享该up主主页的链接中就有

./ 下载到./目录下

-g 上传到谷歌云盘

gd 谷歌云盘的配置名

#### twitter2.py

python twitter2.py

将该文件中的twerID设置为要爬取的推主的id

然后运行即可
